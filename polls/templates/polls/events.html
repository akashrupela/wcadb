WCA-PostGres

Welcome to my website built on WCA Database using Django and PostgreSQL. You can browse following pages:
  • Average Ranks - Top Average rankings for all WCA Events.
  • Single Ranks - Top Single rankings for all WCA Events.
  • Upcoming Competitions - See List and details of all upcoming WCA competitions.
  • Oldest World Records - See list of oldest world records.
  • Sum of Average Ranks - See sum of average Ranks across all events.
  • Sum of Single Ranks - See sum of single Ranks.
  • Sum of 3x3,4x4,5x5 Ranks - See sum of ranks for these specific events.
  • Past Competitions - See all details, rankings, scrambles for past competitions.
  • Best Medal Collection - See best medal tally.
  • Best Podiums - See the best podiums achieved at a competition.
  • Most Persons - See the countries with most participants
  • Most Competitions - See the competitors with most competitions
